Welcome to the HyperSearch documentation!¶

HyperSearch is a Python module to perform a hyperparameter gridsearch for neural network design. HyperSearch can use any of the Python modules Keras, Scikit-learn and Scikit-neuralnetwork to run its hyperparameter search, and can use choice of Python module as another hyperparameter.
