Welcome to the HyperSearch documentation!
=========================================

.. _Keras: http://keras.io/
.. _Scikit-neuralnetwork: http://scikit-neuralnetwork.readthedocs.org/en/latest/index.html
.. _Scikit-learn: http://scikit-learn.org

HyperSearch is a Python module to perform a hyperparameter gridsearch for neural network design. HyperSearch can use any of the Python modules `Keras`_, `Scikit-learn`_ and `Scikit-neuralnetwork`_ to run its hyperparameter search, and can use *choice of Python module* as another hyperparameter.
